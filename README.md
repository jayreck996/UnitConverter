# Unit Converter CLI Tool

A simple command-line tool for converting units of measurement, built with Python.

## Features

* Convert between common units of length (km, miles, meters)
* Convert between temperature units (celsius, fahrenheit, kelvin)
* Easy-to-use command-line interface

## Installation

1. Clone this repository: `git clone https://github.com/jayreck996/unitconverter.git`
2. Navigate to the project directory: `cd unitconverter`
3. Make sure you have Python 3 installed: `python3 --version`

## Usage

```bash
python3 index.py -v <value> -f <from_unit> -t <to_unit>
